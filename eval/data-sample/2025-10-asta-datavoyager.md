# Asta DataVoyager: Data-driven discovery and analysis

**Date:** October 2025  
**Source:** [Allen Institute for AI](https://allenai.org/blog/asta-datavoyager)

## Summary

Asta DataVoyager is a feature within the Asta ecosystem designed to assist scientists in analyzing structured datasets. The tool enables researchers to upload datasets and pose questions in natural language, receiving concise answers, visualizations, and reproducible code.

## Key Features

### Supported Formats
- CSV
- Excel
- JSON
- HDF5
- TSV
- Parquet

### Natural Language Interface
Researchers can ask questions in plain language and receive:
- Concise answers
- Visualizations
- Reproducible code

### Data Privacy
- Uploaded datasets automatically deleted after seven days
- Data is not used for model training
- Maintains strict data privacy standards

## Cancer AI Alliance Integration

The Cancer AI Alliance (CAIA) has implemented a federated version of Asta DataVoyager that:
- Allows AI models to analyze de-identified data across multiple cancer centers
- Maintains data privacy and security
- Aims to uncover new insights into cancer treatments and outcomes

## Access

Asta DataVoyager is available to all users through the "Analyze data" tab in Asta, providing a transparent and user-friendly platform for data-driven discovery.
